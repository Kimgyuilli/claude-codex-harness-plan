# Harness Docs

이 디렉터리는 이제 `v1 실사용 개선` 기준으로만 유지한다.

## 현재 문서 구조

- `V1-ROADMAP.md`
  - v1 목표, 개선 범위, 우선순위
- `TASKS.md`
  - v1 실행 SSOT
  - 지금 할 일, 진행 상태, 개선 체크리스트
- `SESSION-HANDOFF.md`
  - 다음 세션 재개 메모
- `scripts/`
  - 운영 보조 스크립트
- `archive/v0/`
  - v0 설계/검증/측정/운영 기록 전체 보관

## 문서 원칙

1. 루트 문서는 `v1` 활성 문서만 둔다.
2. `v0` 완료 문서는 수정하지 않고 `archive/v0/` 아래에 보존한다.
3. 새 실험, 측정, 회고가 끝나면 활성 문서에 남기지 말고 아카이브 여부를 먼저 판단한다.
4. 정량 측정은 선택이다. 실제 사용 체감이 더 중요한 판단 기준이다.
5. 다음 작업은 항상 `TASKS.md`와 `SESSION-HANDOFF.md`만 보고 시작할 수 있어야 한다.
