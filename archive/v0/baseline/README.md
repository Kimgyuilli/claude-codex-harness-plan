# v0 Baseline Archive

이 디렉터리는 `v0`에서 준비했던 baseline 측정 문서를 보관한다.

## 파일

- `README.md`
  - baseline 측정 목적과 스키마
- `template.md`
  - 측정 템플릿
- `summary.md`
  - 집계용 문서
- `task-b2-hpa-manifest.md`
  - 실제로 기록을 시작했던 baseline 메모

## 현재 의미

v1 기준으로 baseline은 필수가 아니다.
따라서 이 디렉터리는 "예전에 어떤 측정을 하려 했는지"를 보존하는 참고 자료로 본다.

## 언제 참조하는가

- 정량 측정을 다시 도입하고 싶을 때
- 예전 기록 형식이나 스키마를 재사용하고 싶을 때
- baseline을 왜 선택 사항으로 내렸는지 맥락을 확인하고 싶을 때
