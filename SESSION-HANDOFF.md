# Session Handoff — Harness v1

> 역할: 다음 세션 재개 메모

## 현재 위치

- v0 문서는 `archive/v0/`로 내림
- 루트 문서는 이제 `v1` 활성 문서만 유지
- 다음 시작 task: `PeakCart/docs/plans/task-hpa-manifest.md`
- 다음 시작 명령: `/plan task-hpa-manifest`

## 바로 할 일

1. PeakCart 브랜치와 작업 트리 확인
2. `task-hpa-manifest` 계획서 확인
3. `/plan task-hpa-manifest`
4. 이어서 `/work` → `/ship --execute`
5. 사용 체감 메모 작성

## 기억할 사실

- v1의 목적은 실사용 편의 개선이다

- v0 기록은 `archive/v0/`에 보존돼 있다
- v1에서 바꾸는 것은 주로 출력, lint, 자동화 범위 같은 운영 개선이다
